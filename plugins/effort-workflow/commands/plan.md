---
description: Run a decomposition planning session for the current effort.
argument-hint: "[context]"
---

Invoke the `effort-workflow:planning` skill to run a decomposition session.

If `[context]` is provided, use it to focus the session — for example, a specific stub to expand, a tier to work at, or a goal to decompose.

If no context is provided, start by reading `01 TASKS.md` to assess the current decomposition state, then guide a session at the most actionable tier.
